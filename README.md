# 美丽下午茶时光 - 微信小程序

一个专为女性打造的下午茶社交平台，让每一个下午都充满甜蜜与温馨。

## 项目简介

「美丽下午茶时光」是一款基于微信小程序的社交活动平台，致力于为女性用户提供优质的下午茶活动体验。

## 功能特性

- 🎯 **活动广场** - 发现精彩的下午茶活动
- 📋 **活动详情** - 查看活动详情并报名参加
- 👤 **个人中心** - 管理个人信息和活动记录

## 技术栈

- 微信小程序原生开发
- WXML + WXSS + JavaScript
- 响应式设计

## 快速开始

### 环境要求

- 微信开发者工具
- Node.js（可选，用于图标生成）

### 开发流程

1. **克隆项目**
```bash
git clone https://github.com/gloryhonourmyheart/afternoon-tea-miniprogram.git
```

2. **打开项目**
- 打开微信开发者工具
- 选择"导入项目"
- 选择项目目录

3. **预览效果**
- 点击"预览"按钮
- 使用微信扫码查看效果

## 项目结构

```
afternoon-tea-miniprogram/
├── app.js              # 应用入口
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── project.config.json # 项目配置
├── sitemap.json        # 站点地图
├── images/             # 图标资源
├── pages/
│   ├── index/          # 首页-活动广场
│   ├── activity/       # 活动详情页
│   └── profile/        # 个人中心页
└── scripts/            # 辅助脚本
```
